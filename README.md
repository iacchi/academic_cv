# Academic CV

Academic CV is a LaTeX document class with an associated template that aims at simplifying 
the creation of a Curriculum Vitae in LaTeX while retaining a good amount of flexibility for 
what concerns the page structure and its content. Its sections are focussed for an usage by 
academics, but it can be easily adapted to suit other people.

## Features

This class allows to easily change the colours of the page headers and of the first page 
sidebar. There are specific macros that simplify the creation of the content for the 
sidebar in the first page, as well as entries for what concerns education, work experience, 
publications, and conference attendance. Page dimensions and font size can be easily changed 
in the same way that it's usually done for the article class.

The main content is divided in the following sections: Education, Work experience, Teaching 
experience, Funding, Publications, Conference attendance and Scientific leadership. The 
sections can be moved around freely, and more can be added.

Compared to other CV templates out there, especially those with headers or sidebars, the main 
text of this template is not paginated, i.e. you write text normally within the document 
environment without having to break every single page. A single \newpage may be required to 
separate the first page from the rest of the document, to avoid trapping a macro to describe 
work or teaching experience between two pages, with odd results on the margins associated to it.
